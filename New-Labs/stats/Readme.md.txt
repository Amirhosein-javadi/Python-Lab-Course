df.values: This function will give us all the values that are in use in our dataframe
df.columns: This function will show us the name of columns with their type
df.index: This function will show us the name of rows with their type
df.describe: This function will give us different information:
			* The number of Values in each column
			* The mean of values in each column
			* The std of values in each column
			* The minimum of values in each column
			* The maximum of values in each column
